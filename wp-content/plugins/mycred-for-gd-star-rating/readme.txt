== Description ==

[GD Rating System](https://wordpress.org/plugins/gd-rating-system/) ( a.k.a. GD Star Rating) is a powerful, highly customizable and versatile ratings plugin to allow your users to vote for anything you want. This plugin allows you to reward users with points for rating content or content authors for getting content rated.

= myCred for GD Star Rating Features =
Here are the most notable features of myCred for GD Star Rating.

* Award points for rating content.
* Award points for getting content rated.
* Badge support.
* Translation ready.

= Plugin Requirements =

* [myCred 1.8+](https://wordpress.org/plugins/mycred/)
* [GD Rating System](https://wordpress.org/plugins/gd-rating-system/)
* WordPress 5.0+
* PHP 5.3+

== Installation ==

1. Go to Plugins > Add New.
2. Under Search, type myCred for GD Star Rating
3. Find myCred for GD Star Rating and click Install Now to install it
4. If successful, click Activate Plugin to activate it and you are ready to go.
