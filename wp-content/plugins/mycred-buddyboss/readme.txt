Connect mycred with BuddyBoss

== Description ==

**myCred BuddyBoss** is a myCred integrated add-on that allows you to reward myCred points based on BuddyBoss events.

== Why use myCred BuddyBoss? ==

**myCred BuddyBoss** add-on focuses on rewarding points based on triggered hooks associated with BuddyBoss events.

The hooks are pre-configured and cover most-used events on the BuddyBoss platform like **Follow Events**, **Profile Events**, **Friendship Events**, **Group Events**, **Message Events**, **Friendship Events**, **Group Events**. 

Each event, when triggered, will generate a reward notification.

== myCred BuddyBoss Features ==

**General features**
- Reward points based on multiple events of BubbyBoss Platform
- Exclusive Hooks for Dedicated BuddyBoss Platform Features.
- Award or Deduct Points based on the triggered event
- Real-time points notification and log update.

**Follow event hooks**
- Award points to users when they follow another user;
- Deduct points when a user stops following;
- Deduct points when users lose followers;
- Award points to users when they gain a new follower;

**Profile event hooks**
- Award points for changing profile avatar;
- Award points for changing cover image;
- Award points for updating user profiler;
- Award points for meeting minimum profile completion requirements;
- Award points for activating user account;
- Award points when a user is assigned a specific profile type;

**Friendship event hooks**
- Award points to users when other users accept their friend request;
- Award points to users when they send a friendship request;
- Deduct points from users when they lose a friend;

**Group event hooks**
- Award points for creating groups;
- Award points for joining groups;
- Award points for joining specific groups;
- Deduct points for leaving groups;
- Deduct points for leaving specific groups;
- Award points when users are accepted to private groups;
- Award points for sending group invitations;
- Award points to users when they publish a message on the group activity stream;
- Deduct points from users when their message is removed from the group activity stream;
- Award points to users who get promoted;

**Forum event hooks**
- Award points to users when they create a new forum; 
- Award points to users when they create a new forum topic;
- Award points to users when they select a favorite topic;
- Award points to users when they are selected as a favorite author;
- Deduct points when a user deletes a forum topic;
- Award points when a new reply is published;
- Deduct points when a user deletes a forum;
- Deduct points when a user deletes replies on the forum;
- Award points to a user when they create a specific forum;
- Award points to a user when they reply to a specific event;
- Award points to users when they select a favorite from specific topics;
- Award points to users when they select a favorite topic from a specific forum;

**Message event hooks**
- Award points for sending a private message to other users;

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.

== Frequently Asked Questions ==

= How can I activate myCred elements on profile, listing or activities? =

You will find all the settings to manage the elements by navigating to myCred point type > Hooks > BuddyBoss: Profile Events and "BuddyBoss Profile Tab" boxes.

= How can I display user earnings on BuddyBoss activity feed? =

On each type edit screen (points type) you will find setting to manage which elements display on BuddyBoss activity feed.
